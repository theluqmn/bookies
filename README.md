# Bookies - a bookworm's heaven

A website where you can write reviews and attach notes about books you've read
in exchange for points! Use the points to decorate your profile.

## Technical

- users(id, name, password)
- books(id, name, author, description)
- reviews(book_id, user_id, text, datetime, upvotes[], downvotes[])
- notes(book_id, user_id, text, datetime)

- languages: Go, HTML
- libraries: HTMX, echo, Tailwind CSS
- server: HackClub Nest
